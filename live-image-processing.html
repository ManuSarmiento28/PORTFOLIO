Memory as a Multisensory Experience

An installation exploring how memory can be activated through sound, movement, light, and real-time visual systems.

01

Project Concept

This project explores memory as something layered, emotional, and responsive instead of linear. The installation combines visual and auditory stimuli to show how memories can be triggered, distorted, or reconstructed through sensory input.

Tools Used

  • TouchDesigner
  • p5.js
  • Isadora
  • Kinect
  • After Effects
02

My Contribution

Sound-Responsive Visuals

My main contribution focused on creating visuals that respond to music and sound. I explored how audio could affect motion, rhythm, intensity, and atmosphere inside the installation.

TouchDesigner Experiments

I worked with TouchDesigner to create reactive visual systems and sound-driven experiments that translated audio into generative movement and visual changes.

p5.js Interaction

I also developed a p5.js interactive piece using motion and sound behavior, connecting hand movement with visual response.

Technical Setup
